Official brand assets for [Menapia](https://menapia.tech) and the Metsprite drone product line.

## Naming Convention

Files follow the pattern: `{brand}-{element}-{variant}-{color}.{ext}`

- **brand**: `menapia` or `metsprite`
- **element**: `logo`, `icon`, `drone`
- **variant**: `horizontal`, `dandelion`, `animated`, `static`
- **color**: `dark` (teal/black, for light backgrounds) or `white` (for dark backgrounds)

## Usage

These assets are provided for use by partners, press, and stakeholders. Please use them as-is without modification to colours or proportions.
